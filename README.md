**# Text To Speech Using SpeechT5**



This project converts text into speech using Microsoft SpeechT5 model and Streamlit interface.



**## Features**

\- Text to speech conversion

\- AI voice generation

\- Interactive Streamlit UI



**## Tech Stack**

\- Python

\- HuggingFace Transformers

\- PyTorch

\- Streamlit



**## How To Run**



1\. Clone repository

2\. Install dependencies



pip install -r requirements.txt



**3. Run project**



streamlit run app.py
